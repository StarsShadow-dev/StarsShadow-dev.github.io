How to use the SECTION SIGN to change the color and style of text in minecraft!

First you need to get the "section sign" (§) you can either copy and paste it from here.
or if you are on a Mac you can press command, control, space, to open up the character picker... (that isn't very useful because minecraft bedrock does not run on a Mac)

Changing colors:

§O Black
§1 Dark Blue
§2 Dark Green
§3 Dark Aqua
§4 Dark Red
§5 Dark Purple
§6 Gold
§7 Gray
§8 Dark Gray
§9 Blue
§a Green
§b Aqua
§c Red
§d Light Purple
§e Yellow
§f White
§g Gold (Minecoin)

so let's say you want to turn some text red
start by typing §c and then whatever text you want to be red (without a space in between)

Example! §chello becomes:

hello

changing styles:

§k Obfuscated (makes text crazy)
§I Bold
§o Italic


Fun fact you can mix these together just like the hello from earlier:
add Red (§c) and Italic (§o)
To get §c§ohello or...

hello

Final note you occasionally you might want to reset formatting you can do that by using: §r
